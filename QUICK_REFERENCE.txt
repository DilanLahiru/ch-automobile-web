## 📋 Data Being Collected & Sent

The booking form collects and sends to your backend:

```
CUSTOMER INFORMATION:
├─ customerId (email or customer ID)
├─ customerName (full name)
├─ customerContactNumber (phone)
└─ Email

VEHICLE DETAILS:
├─ vehicleNumber (license plate)
└─ vehicleModel (make and model)

APPOINTMENT DETAILS:
├─ appointmentDate (YYYY-MM-DD format)
├─ appointmentTime (HH:MM AM/PM format)
├─ serviceType (dropdown selection)
├─ status (always "pending")
└─ note (customer's additional notes)
```

## ⚙️ How It Works

```
User fills form
    ↓
Form validates required fields
    ↓
User clicks "Confirm Booking"
    ↓
Frontend formats data (dates/times)
    ↓
Sends POST to /api/appointments
    ↓
Backend receives & saves to database
    ↓
Backend returns appointment ID
    ↓
Frontend shows success message
    ↓
Form resets & ready for next booking
```
